Projects I've Worked On

Vehicle Data-Collection and Reporting Web App

I worked with the researchers at EPRI on a data collection project involving remote hybrid vehicles. The data was processed with Apache Spark to create summary reports. The image shows an accompanying web app we built to monitor the fleet. For more information click here.

Dynapower Remote Monitoring System

For Dynapower, we built a remote monitoring system SaaS for power transformers. My main contribution in this project was the graphing system. After failing with SQL, I created a data pipeline to process the graphing data (coming in at thousands of points-per-second) using Amazon Kinesis and a point reduction algorithm. The graphs now load in faster than you can say "Oklahoma."
